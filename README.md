# Peptide Monitor

Peptide Monitor is an anonymous pharmacovigilance platform designed to collect observational peptide usage data from volunteer participants.

## Purpose

The platform enables users to:

- Create anonymous accounts
- Log peptide usage
- Record biometrics
- Record side effects
- Record outcomes
- View anonymised community trends

## Data Principles

- No medical advice provided
- Anonymous participation
- Self-reported observational data
- GDPR compliant data handling
- Commercial research dataset generation
